Commands and Dialog Controls (Check Boxes, Edit Boxes)

Menu Commands

File/Select Folder

Brings up a dialog box for selecting a directory in which the files to be printed reside. The path is displayed on the line labelled "Path".

Options/Calibrate X

Brings up the Select Printer Dialog box. After selecting the printer, a dialog appears with four edit boxes:

  • Number of Characters to print in a line
  • Odd Page Offset (pixels) -- for double side printing
  • Even Page Offset -- determines the left margin
  • A Tweak value -- determines the right margin

A fixed width font is used. The width of the page is determined and the number of characters is used to determine a width for the characters. However printers don't print to the edge of the paper so margins are needed on the left and right to make sure the text is readable.

The offset and tweak values are nominally in pixels but they are rather arbitrary values. So one needs to try some values to see what it all looks like. I usually try a value for the number of characters/line first. If the text is acceptably readable then assign a value to the Odd Page Offset and check the left margin. The command prints a page trial groups;

Width: 95, xTweak: 130, xPage: 10200, xOffset 300, Pixels/Inch: 1200 123456789012345678901234567890123456789012345678901234567890123456789012345678901234567890123456789012345 MMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMMM

The first group of lines displays the current settings. The following groups use the current offset and compute a xTweak value based on the width of the character determined by the width of the page an the number of lines. The xTweak value is computed for each group of lines. Pick the best fit to the page, note the xTweak value and insert it into the dialog box.

Options/Calibrate Y

Brings up the Select Printer Dialog box. After selecting the printer, a dialog appears with three edit boxes:

  • Number of Lines per Page
  • Offset -- determines the top margin
  • yTweak -- a value which controls the bottom maring

The offset and yTweak values are a bit arbitrary because they depend on the printer's pixel density. Start with the number of lines with zero for the offset and yTweak. Run the operation and observe the top and bottom margins. Five pages are output. The first page shows the results of the current values. The next four pages show the results of decreasing the height by one. The line number is printed on each page so that the effect of the change can be determined. The yTweak is printed on the second line of each page. The yTweak value when placed in the dialog box will result in the same output.

Options/Edit Extensions

The program accepts wild cards but can only print text files. The list of extensions acts as a filter of known text files. This command allows the list to be edited.

Help/Introduction

Brings up this Help File.

Help/About

Brings up the About Box that includes some information about the application.

Tool Bar Commands

Select Folder

Brings up a dialog box for selecting a directory in which the files to be printed reside. The path is displayed on the line labelled "Path".

Print Dialog & Print File

Displays the Printer Dialog Box for choosing the printer and print attributes. After the printer is chosen the files that have been chosen by the application from the information provided in the pprint dialog will be printed.

Options/Calibrate X|Y|Edit Extensions

A toolbar drop down list of commands. See above for comments about the individual commands.

About

Brings up the About Box that includes some information about the application.

Exit

Exit the application with no reservations.

Controls

Double Side Output

Check box if the files are to be printed double sided on paper.

Extension Valid

Check box to assert that the file is a text file regardless of its extension

Header is Output

Check box to assert that a header and footer shall be output on each page.

Search Subdirectories

If not checked only the specified directory will be searched for the specified file(s)s. If checked all subdirectories will also be searched for the specified file(s).

Tab Stop Spacing

Tab stops will be determined by the value in the edit box. When tabs appear in the text the next character will appear at the next tab stop after the current position.

Specify Print Only After Date/Time

My favorite command: Only print files requester that have been recently changed as determined by the date. Actually the application does not know if they have been changed recently, it only prints the specified file(s) which have a modified date after the date/time given.

Command Line

This is the place where the file(s) to be printed are specified. Use wild cards '*' and/or '?' if warranted. Separate file specifications with a space.
